Add CSV Import as a Premium feature to Fortis.
Supported brokers: Trading 212, Degiro, Trade Republic, 
Lightyear, Freedom24, Scalable Capital, Flatex, 
Saxo Bank, Revolut, + Generic CSV.
IBKR is excluded for now.

### Trading 212:
"1. Open Trading 212 (web or app)
 2. Go to History tab
 3. Tap the export icon (top right)
 4. Select date range → Export
 5. Upload the downloaded CSV below"

Expected columns:
Action, Time, ISIN, Ticker, Name, 
No. of shares, Price / share, 
Currency (Price / share), 
Exchange rate, Result, 
Total (EUR), Notes

Key columns to parse:
- Ticker:        "Ticker"
- Quantity:      "No. of shares" (sum BUY, subtract SELL)
- Avg Cost:      "Price / share" (weighted average of BUYs)
- Currency:      "Currency (Price / share)"

- Filter:        Action == "Market buy" OR "Limit buy"

## STEP 3: Preview & Confirm

Title: "Review Import"
Subtitle: "X holdings found — review before importing"

### Aggregation logic (applied before preview):
Group all BUY transactions by ticker:
- Total quantity = sum of all BUY quantities
- Avg cost = weighted average:
  avgCost = Σ(qty × price) / Σ(qty)
- First purchase date = earliest BUY date
- Exclude SELL transactions from quantity
  (subtract sells from total qty)
- Skip rows with quantity ≤ 0 after sells

### Preview table:
Each row shows one holding to be imported:

Ticker | Qty | Avg Cost | Date | Status
VWCE   | 10  | €130.50  | Jan 2024 | ✅ New
TDIV   | 25  | €52.00   | Feb 2024 | ✅ New
VWCE   | --  | --       | --       | ⚠️ Duplicate

Status logic:
- "✅ New" — ticker not in existing holdings
- "⚠️ Duplicate" — ticker already in holdings
  Show toggle: "Skip" / "Merge" / "Replace"
  Default: "Skip"

### Merge logic (if user selects Merge):
newQty     = existing.qty + imported.qty
newAvgCost = weighted average of both

### Unknown ticker warning:
For brokers where ticker is derived from ISIN
(Degiro, Scalable, Flatex, Trade Republic):
Show editable ticker field in preview row:
"⚠️ Confirm ticker: [____]"
Pre-fill with best guess if possible.

### Bottom action bar:
"Cancel" (outlined)
"Import X Holdings" (gold, full width)

## POST-IMPORT

After successful import:
- Save all confirmed holdings to SQLite
- Trigger fetchLivePrice() for each new ticker
- Navigate back to Holdings screen
- Show success toast:
  "✅ X holdings imported successfully"

If partial failure:
  "⚠️ X imported, Y failed — tap to review"

---

## ERROR HANDLING

Wrong broker selected (columns don't match):
"This doesn't look like a [Broker] CSV.
 Try selecting 'Generic CSV' instead."

Empty file:
"No transactions found in this file."

All transactions filtered out:
"No BUY transactions found.
 Make sure you exported your trade history."

Unsupported currency:
Flag row in preview with:
"⚠️ Non-EUR price — please verify avg cost"
