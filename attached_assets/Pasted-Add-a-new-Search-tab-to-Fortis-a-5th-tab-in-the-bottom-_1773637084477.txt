Add a new "Search" tab to Fortis — a 5th tab in the bottom navigation.

## TAB
Icon: magnifying glass
Label: "Search"
Position: between Holdings and Performance

---

## SCREEN 1 — Search Home

Header: "Explore"
Subtitle: "Search ETFs, funds & stocks"

Search bar (prominent, top of screen):
- Placeholder: "Search ticker or name..."
- Dark input style, gold focus border
- Minimum 2 characters to trigger search
- Debounce: 300ms
- Clear button (×) when text present

Below search bar (when empty):
Show two sections with horizontal scroll cards:

"Popular ETFs" (static list, preloaded):
VWCE, IWDA, VHYL, CSPX, SWDA, EUNL, TDIV, VAGF

"Major Stocks" (static list, preloaded):
AAPL, MSFT, NVDA, ASML, NOVO-B, SAP, NESN, MC

Each card shows:
- Ticker (bold)
- Current price (fetched live)
- Daily change % (green/red)

---

## SCREEN 2 — Search Results

Triggered when user types 2+ characters.
Call Yahoo Finance search API:
https://query2.finance.yahoo.com/v1/finance/search?q={QUERY}&quotesCount=10&newsCount=0

Each result row:
- Left: Ticker (bold, white) + Type badge (ETF / Stock / Fund)
- Center: Full name (secondary text, 1 line truncated)
- Right: Exchange (muted text)

Filter toggle bar below search input:
[All] [ETFs] [Stocks] [Funds]

Tap result → navigate to Detail screen.

## DATA FETCHING

For detail page, call:
https://query1.finance.yahoo.com/v8/finance/chart/{SYMBOL}?interval=1d&range=1y

Parse from result[0].meta:
- regularMarketPrice
- previousClose
- fiftyTwoWeekHigh
- fiftyTwoWeekLow
- regularMarketVolume
- averageDailyVolume3Month
- currency
- exchangeName

Parse from result[0].summaryDetail (if available):
- trailingAnnualDividendYield
- totalAssets (AUM for ETFs)
- trailingPE (for stocks)

## ERROR HANDLING
- If data unavailable: show "—" never crash
- If chart data empty: show "Chart unavailable"
- If price stale (market closed): show "Closed · Last €XX.XX"
- Network error: show retry button
