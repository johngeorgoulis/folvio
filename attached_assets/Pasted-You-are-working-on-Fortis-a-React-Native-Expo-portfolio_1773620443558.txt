### Layout:

Section 1 — Portfolio Value Chart

- Time range selector: 1W | 1M | 3M | 1Y | All
- Data source: snapshot history (see data model below)
- If < 7 days of data → show message "Keep tracking to see your history"

Section 2 — Return Metrics
Display as cards in a 2x2 grid:

| Card             | Value                        |
|------------------|------------------------------|
| Total Return     | +€X,XXX.XX (+XX.XX%)         |
| Time in Market   | X months                     |
| Best ETF         | VWCE +XX.XX%                 |
| Worst ETF        | CSBGE7 -X.XX%                |

### Total Return calculation:
totalReturn = currentPortfolioValue - totalInvested
totalReturnPct = totalReturn / totalInvested × 100

totalInvested = SUM(holding.quantity × holding.avg_cost_eur) for all holdings

### Per-ETF return:
etfReturn = (currentPrice - avgCost) / avgCost × 100

Section 3 — Benchmark Comparison (Premium only)
- Toggle: "Compare to VWCE" (default benchmark)
- Show portfolio line vs VWCE line on same chart
- VWCE benchmark data: fetch from Yahoo Finance (VWCE.DE)
  using same priceService.fetchHistoricalPrices()
- If free tier: show blurred chart with "Upgrade to Premium" overlay

Section 4 — Dividend Estimate (annual)
- For each holding, estimate annual dividend:
  annualDividend = holding.quantity × holding.priceEUR × (yieldPct / 100)
- Yield % is stored manually by user (no auto-fetch in v1)
- Total estimated annual income in EUR
- Show as: "Estimated annual income: €XXX.XX"
- Small disclaimer: "Based on trailing yield. Not guaranteed."

## SNAPSHOT HISTORY (for chart)

Add to SQLite schema:

portfolio_snapshots table:
- id (integer primary key)
- snapshot_date (text, YYYY-MM-DD, unique)
- total_value_eur (real)
- total_invested_eur (real)
- created_at (text)

Snapshot logic:
- Take one snapshot per day, on app open
- Check if today's snapshot exists → if yes, skip
- If prices available → save snapshot
- If prices unavailable → skip (don't save incomplete data)
- Keep max 365 snapshots (delete oldest if exceeded)

Add to holdings table:
- yield_pct (real, nullable)    -- trailing dividend yield, user-entered

## HISTORICAL PRICE FETCHING

Add to priceService.ts:

fetchHistoricalPrices(symbol, range) → Promise<PricePoint[]>

Endpoint:
https://query1.finance.yahoo.com/v8/finance/chart/{SYMBOL}?interval=1d&range={RANGE}

Range mapping:
"1W" → "5d"
"1M" → "1mo"
"3M" → "3mo"
"1Y" → "1y"
"All" → "5y"

Parse response:
- timestamps: result[0].timestamp (array of Unix timestamps)
- prices: result[0].indicators.quote[0].close (array of closing prices)
- Zip timestamps + prices into PricePoint[]
- Filter out null values (market holidays)

interface PricePoint {
  date: string      -- ISO 8601 date
  priceEUR: number  -- converted if needed (same FX logic as live price)
}
